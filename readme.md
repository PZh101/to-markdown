# 使用简介

## 环境

> windows 11 ; nodejs 18.19.0

## 结构

- kmToMarkdown.cjs 百度脑图转markdown
- xmind.cjs xmind转markdown

## 使用方式

使用kmToMarkdown.cjs时不需要额外的三方库，如需使用xmind.cjs时，请先

```bash
npm install 
```

运行脚本

```bash
node kmToMarkdown.cjs -f yourfile.km

```

或

```bash
node xmind.cjs -f yourfile.xmind

```
